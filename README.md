# lilygo-t-hmi

- Here is a printable case/enclosure for a Lilygo t-hmi. I guess this must be a niche device as I could not find an existing 3d model anywhere, so had to create one. It has cut outs for the buttons, sd slot and pin sockets, and plenty of space for a fat USB-C cable.
- Here is also a demo app which shows a joke of the day. Refresh (or wake) by pressing the screen. The screen will sleep after 10 minutes.

# Flashing the device
```
pio run -e HMI -t upload
```
(or use the Arduino IDE)
